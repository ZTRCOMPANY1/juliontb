Arcade ZT

Um site com vários mini games direto no navegador, feito em HTML + CSS + JavaScript puro, sem NPM e sem JSX. Ideal para hospedar no GitHub Pages como um único arquivo ou dividido em páginas depois.

Como publicar no GitHub

  • Crie um repositório no GitHub.
  • Salve este arquivo como index.html.
  • Envie o arquivo para o repositório.
  • Ative o GitHub Pages nas configurações.
  • Selecione a branch principal e a pasta raiz.
